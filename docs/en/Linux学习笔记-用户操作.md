# Linux Study Notes - User Operations

## Basic Operations

### Add User

```shell
useradd -m username
```

### Set Password

```shell
passwd username
```

### Delete User

```shell
userdel -r username
```

### Delete User Directory

```shell
rm -rf username
```

### Switch Current User

```shell
su username
```
