# Escape Characters

| Escape Sequence | Character                                  |
| --------------- | ------------------------------------------ |
| `\b`            | Backspace                                  |
| `\f`            | Form feed                                  |
| `\n`            | New line                                   |
| `\r`            | Carriage return                            |
| `\t`            | Horizontal tab                             |
| `\unnnn`        | Unicode character                          |
| `\xnn`          | ASCII character, where nn is a hexadecimal |
| `\'`            | Single quote                               |
| `\"`            | Double quote                               |
| `\\`            | Backslash                                  |
