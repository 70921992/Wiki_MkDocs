# \ تسلسل الهروب

| تسلسل الهروب | الحرف |
| -------- | ------------------------------------------ |
| `\b`     | حرف العودة |
| `\f`     | حرف الصفحة |
| `\n`     | حرف السطر الجديد |
| `\r`     | حرف العودة إلى الوراء |
| `\t`     | حرف التبويب |
| `\unnnn` | حرف يونيكود |
| `\xnn`   | حرف ASCII ، يتم تحديد رمز الحرف بواسطة أرقام ست عشرية nn |
| `\'`     | علامة اقتباس مفردة |
| `\"`     | علامة اقتباس مزدوجة |
| `\\`     | حرف الانعكاس الخلفي الفردي |
