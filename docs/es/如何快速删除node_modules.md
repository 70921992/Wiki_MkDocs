# Cómo eliminar rápidamente node_modules

Instale globalmente `rimraf`:

```shell
npm install rimraf -g
```

Cambie al directorio raíz que contiene la carpeta `node_modules` y ejecute el siguiente comando:

```shell
rimraf node_modules
```

¡Y eso es todo!
