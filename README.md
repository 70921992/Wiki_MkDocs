# Wiki_MkDocs

个人 Wiki（基于 MkDocs）

Check the version of MkDocs: `mkdocs --version`  
Upgrade MkDocs: `pip install -U mkdocs`  

Check the version of Material theme: `pip show mkdocs-material`  
Upgrade Material theme: `pip install --upgrade mkdocs-material`
